Информация

		
Горячии клавиши

		
		
Работа с текстом:

		
Ctrl + C (с) - копировать выделенный текст в буфер обмена

		
Ctrl + V (м) - вставить текст из буфер обмена на место курсора

		
Ctrl + X (ч) - вырезать выделенный текст в буфер обмена

		
Ctrl + Z (я) - отменить последнее действие

		
Ctrl + Y (н) - отменить отмену

		
Ctrl + A (ф) - выделить всё

		
		
Для программирования:

		
Esc + стрелка вверх - вставляет команду "вверх"

		
Esc + стрелка вправо - вставляет команду "вправо"

		
Esc + стрелка вниз - вставляет команду "вниз"

		
Esc + стрелка влево - вставляет команду "влево"

		
Esc + пробел - вставляет команду "закрасить"

		
		
Shift + F9 - быстрое выполнение (без анимации)

		
F9 - обычное выполнение (с анимацией)

		
Esc - остановить выполнение

		
F2 - шаг (выполнить одну команду)

		
F4 - выполнить до курсора

		
F8 - форматировать код

		
Tab - добавляет отступ для выделеного текста

Возможности WebКумир

		
Операторы 'не', '**', '*', '/', '+', '-', '<=', '>=', '<>', '=', '<', '>', 'и', 'или';
Обявление и присваивание в одной строке (цел ц=21,вещ в=0.204,лит л='600',сим с='с',лог ло=да);
Присваивание (имя:=выражение);
Циклы:
  нц
  |
  |-->для
  |   |
  |   |-->i от i1 до i2
  |   |
  |   |-->i от i1 до i2 шаг i3
  |
  |-->пока условие
  |
  |-->формула раз
  |
  |-->тело_цикла
  ...
  кц или кц_при

выбор:
  выбор
	при условие 1: серия 1
	при условие 2: серия 2
	...
	при условие n: серия n
	иначе серия n + 1
  все

  выбор
	при условие_1: серия_1
	при условие_2: серия_2
	...
	при условие_n: серия_n
  все    

Условия:  
  если условие
	то серия1
	иначе серия2
  все

  если условие
	то серия1
  все

Команды:
  выход
  имя_алгоритма - процедуры
  имя_алгоритма - процедуры(список_параметров_вызова)
  вывод имя - 1,...,имя - N
  ввод имя - 1,...,имя - N

Константы:
  МЦЕЛ = 2147483647
  МВЕЩ = 1.7976931348623157e+308
  да, нет - Логические константы
  нс - перевод строки
